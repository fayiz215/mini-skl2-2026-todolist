# AGENTS.md

## Project

- **todo-app/** - React + Vite + Vanilla CSS To-Do List app
- Run with `npm run dev` inside `todo-app/`
- Build with `npm run build` (output to `todo-app/dist/`)

## Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm run lint` | ESLint check |

## Tech Stack

- React 19 + Vite 8
- Vanilla CSS (no framework)
- ESLint for linting
